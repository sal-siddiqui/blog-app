# blog-app

A blogging web application built with Flask and Bulma. I am following the [Python Flask Tutorial](https://www.youtube.com/playlist?list=PL-osiE80TeTs4UjLw5MM6OjgkjFeUxCYH) by [Corey Schafer](https://www.youtube.com/@coreyms).

## Run the App

To run the application, you can either clone the GitHub repository to your local machine or pull the image (salsiddiqui02/flask-blog) from Docker. Thereafter, navigate to `localhost:5000/auth/login`.
